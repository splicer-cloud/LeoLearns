# Leo Learns!

Phone-first reading practice prototype focused on phonics and high-frequency words.

## Current structure

- `Home`: explains the learning flow and shows the current skill
- `Skills`: a library of phonics lessons
- `Lesson`: one phonics pattern at a time with examples, sentence practice, and Pattern Check
- `Words`: simple high-frequency word cards with audio comparison
- `Parent`: edit and save the word list for the word-card page

## Current phonics library

The app now includes a larger set of phonics lessons, including:

- short vowels
- sneaky h teams
- beginning blends
- ending blends
- ck/ff/ll/ss endings
- magic e
- bossy r
- vowel teams
- tricky y
- glued sounds
- oi/oy
- ou/ow
- soft c and soft g
- ending le
- oo sounds
- aw/au
- kn/wr
